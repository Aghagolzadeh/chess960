# Documentation

This folder explains how the repo is organized and how to extend the Chess960 research platform.

- `repo_navigation.md`: where code belongs and how to add new pieces.
- `chess960_overview.md`: Chess960 rules, starting positions, and castling notes.
- `engines_overview.md`: engine interfaces and baseline behavior.
- `rl_algorithms.md`: reinforcement learning lesson and current training limits.
- `experiments.md`: arena runs, logs, and reproducibility.
- `learning_audit.md`: RL audit findings, fixes, sanity checks, and success criteria.
